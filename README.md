# ReconX

Advanced recon toolkit for subdomains, JS endpoints, historical URLs, and HTTP checks.

## Features
- Subdomain Enumeration
- JS Recon (endpoints from JS + inline)
- Historical Recon (Wayback, filtered)
- HTTP Security Analysis

## Install
```bash
git clone https://github.com/SaurabhSingh773/ReconX.git
cd ReconX
pip install -r requirements.txt

## Usage
